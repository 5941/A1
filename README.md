# A1
=======
# 项目使用相关技术
* nodejs
* angualrjs
* gulp
* webpack

# JS语法
目前使用的是ES6,当然也可以使用ES5


##项目启动

```
##克隆项目
git clone git@github.com:a1cloud/startkit.git
cd startkit

## 安装项目依赖模块
npm i

## 安装项目构建工具gulp
npm i gulp -g

## 启动项目开发模式
gulp
## 生成生成环境包
gulp build --release

## 安装部署工具pm2
npm install pm2 -g

##部署程序
pm2 deploy deploy.json dev
pm2 deploy deploy.json production
pm2 deploy deploy.json test

```
